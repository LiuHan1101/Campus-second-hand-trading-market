## 项目简介  
本项目旨在开发一个便捷、安全、高效的校园二手商品交易平台，为校内学生提供闲置物品发布、浏览、交易与交流的一站式服务，促进资源循环利用，营造绿色、经济的校园生活氛围。

## 主要功能（暂拟）  
- 用户注册与身份认证（校内验证）  
- 商品发布与管理（图文、价格、分类等）  
- 商品浏览与搜索（分类筛选、关键词查询）  
- 在线沟通与交易协商  
- 交易评价与信誉系统  
- 后台管理与数据统计
